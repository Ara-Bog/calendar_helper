# Очередная недоделка

Данный проект представляет собой простую форму для расчета рабочих смен при сменном графике. Да, таких много, но для 7.8 нету или платно.

# Сделано:

-   [x] Автозаполнение рабочих часов;

-   [x] Расчет больничных и отпусков;

-   [x] Графики 1/1, 2/2, 3/3, кастомный;

-   [x] Загрузка/выгрузка готового графика.
